# tencent-mapsdk-samples-for-android
#地图SDK示例中心
##设计目的：
帮助用户快速实现需要多种接口组合而成的较为复杂的业务场景需求。

#示例中心模板
##包含内容：
1.示例介绍
2.示例视频
3.应用场景
4.产品及接口名称
5.示例核心内容讲解
6.核心代码
7.源代码链接
8.demo下载链接


## 设置 key
用户可以在 demo 的清单文件中定义

```xml
<meta-data
    android:name="TencentMapSDK"
    android:value="您的 key" />
```

也可以使用工程里的方式在本地工程中创建 local.properties 并添加：

```
mapsdk.key=您的 key
```
